Primeiros passos com HTML

Tags h1, h2, h3, h4, h5 e h6

Essas tags representam seis níveis de título de seção. O "h1" é o nível mais alto e o "h6" é o nível mais baixo.

Tag p

Essa tag representa um parágrafo. Em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo
como imagens, e campos de um formulário.

Tag mark

Essa tag representa um trecho em destaque em um texto, por exemplo, uma sequência de texto marcado como referência,
devido à sua relevância em um contexto particular.

Tag small

Essa tag é utilizada para definir um pequeno texto dentro da página HTML.

Tag i

Essa tag define um texto em itálico. É de grande valor e utilidade para marcar, termos técnicos, termos de outras
linguagens, etc.

Tag u

Essa tag é utilizada para sublinhar textos.

Tag strong

Essa tag é utilizada para deixar uma palavra ou parte de um texto em negrito.

Tag ol

Essa tag define uma lista ordenada.

Tag ul

Essa tag define uma lista não ordenada.

Tag li

Essa tag define os itens dentro de uma lista.

Tag a

Essa tag define um hyperlink, podendo levar a outra página ou para alguma parte da mesma página ao ser clicado.

Tag hr

Essa tag define uma linha horizontal, muito utilizada para fazer separações e quebras de linha.

Tag sub

Essa tag define um texto subscrito.

Tag sup

Essa tag define um texto sobrescrito.

Tag blockquote

Essa tag é utilizada para fazer uma citação longa.
